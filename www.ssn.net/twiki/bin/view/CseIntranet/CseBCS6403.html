Course Notes

Unit - I & 5 (SOFTWARE PROCESS AND PROJECT MANAGEMENT)

S.NO Topic Material, Summary and Assessment
1 Introduction to Software Engineering
2 Software Process - Prescriptive, Specialized  
3 Estimation:LOC and FP Estimation,COCOMO-II, Make/Buy Decision
4 Project Plan & Planning process
5 Project Scheduling & Tracking  
6 RFP Risk Management-Identification, Projection RMMM  
7 Process and project metrics

Unit - 2 (REQUIREMENT ANALYSIS AND SPECIFICATION)

S.NO Topic Material, Summary and Assessment
1 Software Requirement
2 Software Requirement Engineering Process
3 Classical Analysis

Unit - 3 (SOFTWARE DESIGN)

S.NO Topic Outline, Material, Assessment
1 Design : Process, Concepts, Model & Heuristic
2 Architectural Design & Architectural styles
3 User Interface Design: Interface analysis, Interface Design
4 Component level Design: Designing Class based components, traditional Components

Unit - 4 (TESTING AND IMPLEMENTATION)

S.NO Topic Outline, Material, and Assessment
1 Software testing fundamentals-Internal and external views
2 White box testing- basis path testing-control structure testing
3 Black box testing
4 Regression, Unit, Integration, System testing and Debugging
5 Software Implementation Techniques : Coding practice, Refactoring
