EC6504

Microprocessors and Microcontroller

Staff in-charge :: K. R. Sarath Chandran:: Assistant Professor / CSE

Syllabus

Lesson Plan

Name List

OBJECTIVES:
The student should be made to:
1.Study the Architecture of 8086 microprocessor.
2.Learn the design aspects of I/O and Memory Interfacing circuits.
3.Study about communication and bus interfacing.
4. Study the Architecture of 8051 microcontroller.

OUTCOMES:
At the end of the course, the student should be able to:
1.Design and implement programs on 8086 microprocessor.
2. Design I/O circuits.
3. Design Memory Interfacing circuits.
4. Design and implement 8051 microcontroller based systems.

Notes

Topic Content
Unit 1
8086 - Basics
Architecture and Pins
Addressing modes and Memory
Instruction Set and Assembler directives
BIOS and DOS Interrupts
Unit 2
Maximum and Minimum modes
Multiprocessor Configurations
8087 - Numeric Data Processor
8089 - I/O Processor
Unit 3
8255 - Programmable Peripheral Interface
8254/8253 - Programmable Interval Timer
8279 - Keyboard and Display Controller
8259 - Programmable Interrupt Contoller
8237 - Programmable DMA Contoller
8251 - USART
Unit 4 and 5
8051 - Microcontroller
8051 - Instruction Set
K/B Interfacing with 8051
LCD Interfacing with 8051
DAC Interfacing with 8051
ADC Interfacing with 8051
